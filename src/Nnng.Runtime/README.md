During the packaging process, nng binary archives are automatically downloaded from the [nng-ci-build](https://github.com/myd7349/nng-ci-build) release page and then packed into the corresponding NuGet packages named `Nnng.Runtime.[RID]`.

[^1]: This package is compatible with both Intel-based and Apple Silicon (M1, M2, etc.) Mac computers, effectively serving as a combined package of both `osx-arm64` and `osx-x64`.
